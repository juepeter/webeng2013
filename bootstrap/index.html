Probleme der ersten Kurseinheit
ProblemLösung
In Windows 7 Oracle VM Virtual Box Manager nicht als Administrator installiert, dann treten bei Start der VM Probleme Deinstallieren und neu als Admin installieren
USB 2.0-Problem: USB 2.0 is currently enabled for this virtual machine. However, this requires the Oracle VM VirtualBox Extension Pack to be installed. Einfache Lösung: USB 2.0 deaktivieren (Häkchen bei Eigenschaften der VM, USB für USB 2.0 wegnehmen) Grundsätzliche Lösung: Oracle VM VirtualBox Extension Pack installieren. Bei der Erstellung der VM muss das Extension Pack installiert gewesen sein, sonst lässt sich das Häkchen nicht setzen.
PAE-Problem: The guest is trying to switch to the PAE mode which is currently disabled by default in VirtualBox. Eigenschaften der VM: System -> Processor Häkchen bei Enable PAE setzen.
Schlechte Auflösung der Videos Einstellung auf 720p
Webverzeichnis des Webservers, das im Video angegeben wird, existiert nicht Das angegebene Webverzeichnis ist falsch, korrekt ist das Verzeichnis /home/developer/public_html
@ auf Tastatur funktioniert nicht. Dies scheint tastaturspezifischer Fehler zu sei. Standardmässig ist Apple-Tastatur in der VM eingestellt Eine simple Lösung ist das Durchprobieren aller Tasten mit AltGr bis man @ findet. Scheint bei fast allen die Kombination AltGrL zu sein. Alternativ kann man auch die Tastatureinstellung ändern (Achtung: wird erst wirksam nach Neustart der VM oder vielleicht reicht auch Neustart der Terminalsession). Der Hersteller meiner Tastatur ist Trust und ich habe willkürlich eine Trust-Tastatur unter Angeboten ausgewählt und es hat funktioniert. Bei anderen hat Tastatur Microsoft Office oder schlicht Tastatur Deutsch funktioniert. Anscheineńd muss man einfach Probieren.
